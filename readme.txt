CPSC 485 Edit Distance Programming Assignment
Author: Jack Loague
Language Used: C++

How to run the program on Windows:
1) Make sure MinGW is installed (Download: https://sourceforge.net/projects/mingw/)
2) In the command prompt, navigate to the directory of the EditDistance.cpp file.
3) In the command prompt, run the command "g++ EditDistance.cpp -o EditDistance".
4) Run the exe file created by simply running the command "EditDistance."
